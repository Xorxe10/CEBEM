{ LESS }



¿Qué es?

LESS es una extensión de CSS basada en JavaScript. Su sintaxis es prácticamente idéntica a la de CSS, por lo cual es muy sencillo aprenderlo e implementarlo en nuestro método de trabajo, pero añadiendo una serie de nuevas funcionalidades muy interesantes y que aumentan la productividad del usuario. Es un lenguaje compilado, en el cual un archivo JS se encarga de la conversión del código LESS en CSS tradicional.



Principales características

Variables

Se permite el uso de variables, de forma que podemos almacenar datos como códigos hexadecimales de color, números de píxeles, y un largo etcétera, e implementarlos en nuestro código de manera rápida y eficiente. 

@aguamarina: #02C8A7;

.nav{
    background-color: @aguamarina;
}



Mixins

Los mixins hacen referencia a la posibilidad de reutilizar una porción de código ya escrita. Actúa como una especie de herencia, como podemos ver en el ejemplo. 

.espacio1{ /* Generamos una clase */
    border-style: solid;
    border-color: white;
    border-radius: 15px;
}

.espacio2{
    background-color: gray;
    .espacio1(); /* Hacemos referencia a la clase para 'heredar' sus atributos;
}

Anidamientos

LESS nos brinda la posibilidad de anidar código, en vez de escribirlo en cascada. En CSS tradicional, si queremos hacer referencia a un elemento dentro de otro elemento, lo hacemos en cascada, de esta manera: 

.menu{
    background-color: #blue;
}
.menu li{
    color: white;
}
.menu li a{
    text-decoration: none;
}

En LESS se hace más cómodo y legible el anidar el código de este modo:

.menu{
    background-color: #blue;
    li{
        color: white;
        a {
            text-decoration: none;
        }
    }
}

Funciones

LESS incluye una serie de funciones que hacen acciones sobre las propiedades, como saturar color, iluminar, retornar un porcentaje... También soporta operaciones matemáticas. Listado de funciones. 

@width: 800px;
@height: 100px;
.nav{
    width: percentage(@width); /* devuelve el 50% de la variable */
    height: @height + 100; /* soporta operaciones matemáticas */
}
